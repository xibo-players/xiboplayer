# @xiboplayer/player Documentation

**Meta package - installs all Xibo Player packages.**

## Overview

Convenience package that installs all player components:

```bash
npm install @xiboplayer/player
```

## Includes

- `@xiboplayer/core`
- `@xiboplayer/renderer`
- `@xiboplayer/cache`
- `@xiboplayer/schedule`
- `@xiboplayer/xmds`
- `@xiboplayer/xmr`
- `@xiboplayer/sw`
- `@xiboplayer/utils`

## Usage

```javascript
import { PlayerCore } from '@xiboplayer/core';
import { RendererLite } from '@xiboplayer/renderer';
import { CacheManager } from '@xiboplayer/cache';

// All packages available
```
